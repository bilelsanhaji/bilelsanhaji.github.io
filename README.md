# bilelsanhaji.github.io

A minimal static site template intended for publishing with GitHub Pages.

## Structure

- index.html — main entry
- assets/css/styles.css — styles
- assets/js/main.js — client JavaScript
- sitemap.xml, robots.txt — SEO support

## Deploy

This repository is intended to be served from the repository's main branch via GitHub Pages (username.github.io).

To customize, edit index.html and the files in assets/ and push to main.
