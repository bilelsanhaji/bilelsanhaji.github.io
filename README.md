# bilelsanhaji.github.io

Template statique minimal pour GitHub Pages.

Instructions rapides :
1. Vérifie/édite les fichiers (index.html, assets/).
2. Si tout est OK, merge la branche `site-template` dans `main` ou pousse directement sur `main`.
3. Dans les settings du repo > Pages : sélectionner la branche et le dossier (root) si besoin.

Personnalisation recommandée :
- Remplacer les textes, images (`/assets/images/`), et métadonnées OG.
- Ajouter `CNAME` si tu veux un domaine personnalisé.
